# LojaPy
Sistema de loja que simula a interação com o usuário, onde ele pode cadastrar produtos e realizar o fluxo de compra.

Desenvolvi esse projeto junto ao curso de python da Geek University para praticar os conhecimentos adquiridos e adicionei algumas incrementações diferentes. O programa simula um sistema de loja em que executa um menu que lista algumas opções para o usuário prosseguir, sendo possível cadastrar, listar e comprar produtos, além de visualizar o carrinho de compras, finalizar o pedido e fechar a aplicação.
Ao realizar o pedido de compras é apresentado a listagem de produtos disponíveis, podendo selecionar o produto por seu código e inserindo a quantidade desejada, deve ser checado se há a quantidade disponível, e caso tenha o produto é adicionado ao carrinho de compras e é subtraído o valor no estoque e atualizado.
Ao finalizar o pedido deve ser apresentado ao usúario os produtos do carrinho, seus respectivos preços e o valor total da compra de acordo com a quantidade e o preço de cada produto inserido, encerrando a interação e iniciando uma nova.
